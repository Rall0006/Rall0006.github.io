5 HTML tags

Anchor (a)
The anchor (a) tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the anchor element is the hyperlink reference (href) attribute, which indicates the link's destination. Anchor tags can also be used to link to a different section on the same page.
Thematic breaks (hr)
Thematic breaks are useful little lines that help to seperate logical sections of your page. This page has 2 breaks.
List tags (ul, ol, and dl)
List tags help you to easily make bullet points. There are unordered lists (ul), ordered lists (ol), and description lists (dl) like this one.
Images (img)
Images are digital pictures that improve the look of your web page. The source (src) property specifies the picture to display and the alternate text (alt) property allows you to define a title for those who can't see the image.
Headers (h#)
HTML headings are titles or subtitles that you want to display on a webpage. The number beside the little h signifies which level of header you are on. For example, h1 would be the main header of the page and h2 would be a sub-header.
