Let's talk about Bootstrap (no, not the pirate)

Bootstrap is an exccelent tool for building a website quickly if you know how to use it.
It esentially does a half of the formatting work for you by inserting some refference links into the code that connect to premade css code.
Bootstrap has many wonderful tools and features that make it a good buddy for website coding projects.
let's explore some of them together!

It's mobile first design.

Bootstrap not only allows for but focuses on a mobile first development approach.
It does this by using a 12 column flexbox grid in tandem with 5 different responsive tiers.

It's open source!

It's open source which means it's publically available and free to use.
it's made to help everyone from a beginner who wants to try their hand at making a website or an experinced designer who just wants ahelping hand to save them time.
Use Bootstrap's toolkit for HTML,JS, and CSS to help you confindently build your websites and apps in half the time.

Hansel and Gretel and the code.

Bootstrap includes the wonderful feature of breadcrumbs.
Breadcrumbs allow you to see the page you're working on in relation to the hierachy of the navigation.
This can be very useful for seeing how things are connected when building bigger, more complex websites and apps.
but even seing how small 3 page sites connect together can help a lot.

Bootstrap can be an easy to use friend to any coder who decides to embrace it.
Just make sure to use it properly, afterall, with great power comes great resposibility.
